# Amazon Clone – Project Brief

## Project Name
Amazon Clone – Full Stack E-Commerce Web Application

## Repository
https://github.com/Chaitanya8822/amazon

## Author
Chaitanya

## Overview
A production-inspired full-stack e-commerce platform built with Django (Python), HTML, CSS, and Bootstrap. Mirrors core Amazon functionality including user auth, product catalog, cart management, and order processing.

## Core Goals
1. User registration, login, profile management with secure auth
2. Product catalog with categories, search, and filtering
3. Session-based shopping cart (add/remove/update qty)
4. Checkout and order history with status tracking
5. Admin panel for product & order management
6. Clean, responsive UI using Bootstrap 5

## Scope
- Backend: Django (MVT architecture)
- Frontend: Django Templates + Bootstrap 5 + Custom CSS/JS
- Database: SQLite (dev), PostgreSQL-ready (prod)
- Auth: Django's built-in auth + custom UserProfile model
