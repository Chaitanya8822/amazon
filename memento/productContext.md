# Product Context

## Why This Project Exists
Demonstrates mastery of full-stack web development with Python/Django in a real-world, recognizable domain (e-commerce). Suitable as a portfolio project and learning reference.

## Problems It Solves
- Teaches Django MVT architecture in a practical context
- Shows user auth, session management, DB design
- Illustrates cart & order workflows
- Provides a deployable, extensible codebase

## How It Should Work
1. Visitor lands on homepage → sees featured products & categories
2. User registers/logs in → gets a personalized experience
3. Browse products → filter by category, search by keyword
4. View product detail → add to cart
5. Cart page → review items, update qty, remove
6. Checkout → enter address, place order
7. Order history → view past orders and status
8. Admin → manage products, categories, orders

## User Experience Goals
- Fast, clean UI resembling Amazon's style
- Mobile-responsive layout using Bootstrap 5
- Intuitive navigation with search always accessible
- Clear feedback on cart/order actions (success/error messages)
